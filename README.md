## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`

## Dart / Flutter Version

An experimental Dart implementation of the Aura interface is available in `dart_version/main.dart`.

To run the Flutter version:
1. Ensure you have the [Flutter SDK](https://flutter.dev/docs/get-started/install) installed.
2. Create a new Flutter project.
3. Use the code from `dart_version/main.dart` as your `lib/main.dart`.
4. Run `flutter run` to launch the application.
